This R script is used for the statistical evaluation and visualization
of the flood prediction results generated by the deep learning workflow
in the Smart SH Emergency project. The Python-based modelling workflow
produces several CSV files containing performance metrics and
threshold-dependent evaluation results for the flood depth predictions.

The R environment is used to load these exported evaluation datasets,
process them into interpretable formats, and generate publication-ready
plots and tables. In particular, the script compares raw model
predictions with post-hoc calibrated predictions using different
evaluation metrics such as F1 score, Intersection-over-Union (IoU), and
spatial flood extent differences.

The workflow therefore focuses on three main tasks:

-visualization of threshold sensitivity for flood classification
metrics,

-comparison of spatial prediction accuracy between raw and calibrated
predictions,

-generation of formatted statistical tables for documentation and
presentation.

The results are exported as figures and tables that can be used directly
in the written report or presentation.

# visualization: treshold-Sensitivity

``` r
th <- threshold %>%
  mutate(
    threshold_m = as.numeric(thr_m),
    variant     = method,                 # RAW vs CAL_QM
    Pred_extent_m2 = as.numeric(Area_pred_m2),
    Ref_extent_m2  = as.numeric(Area_ref_m2),
    FP_m2 = as.numeric(FP),              
    FN_m2 = as.numeric(FN),
    TP_m2 = as.numeric(TP),
    PredRef_pct = 100 * Pred_extent_m2 / (Ref_extent_m2 + 1e-9),
    FPRef_pct   = 100 * FP_m2 / (Ref_extent_m2 + 1e-9),
    FNRef_pct   = 100 * FN_m2 / (Ref_extent_m2 + 1e-9),
    TPRef_pct   = 100 * TP_m2 / (Ref_extent_m2 + 1e-9)
  )

# llong for plotting
th_long <- th %>%
  select(threshold_m, variant, PredRef_pct, FPRef_pct, FNRef_pct) %>%
  pivot_longer(cols = c(PredRef_pct, FPRef_pct, FNRef_pct),
               names_to = "metric", values_to = "value")

ggplot(th_long, aes(x = threshold_m, y = value, color = variant, group = variant)) +
  geom_line(linewidth = 1) +
  geom_point(size = 2) +
  facet_wrap(~ metric, scales = "free_y",
             labeller = as_labeller(c(
               PredRef_pct = "KI/GRASS Extent (%)",
               FPRef_pct   = "FP / Ref Extent (%)",
               FNRef_pct   = "FN / Ref Extent (%)"
             ))) +
  scale_x_continuous(trans = "log10", breaks = c(0.01, 0.05, 0.1)) +
  labs(x = "Threshold (m) [log]", y = "Prozent", color = "Methode",
       title = "UG2: Threshold-Sensitivität – RAW vs CAL") +
  theme_minimal(base_size = 12)
```
